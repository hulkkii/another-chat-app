# Frontend Structure

The React client lives in the `anotherchatapp.client` folder. It was bootstrapped with Vite using the React + TypeScript template.

Key libraries configured:

- **Tailwind CSS** for styling with a custom `tailwind.config.js` and `postcss.config.js`.
- **TanStack Router** for routing.
- **React** 19 with TypeScript.

The chat interface is currently mocked and can be started with:

```bash
cd anotherchatapp.client
npm run dev
```
